**custom-built Convolutional Neural Network (CNN)** and supports **real-time detection via OpenCV** in Google Colab.

## Features
- Custom CNN trained on UTKFace dataset  
- Dual output heads (Age regression + Gender classification)  
- Real-time camera demo (Open Cv)  
- Huber + Binary Crossentropy loss combo  
- Mean Absolute Error (MAE) used for evaluation
